## Summary

- Convert TSo from WinForms (WebView2) to .NET 8 Console App backend API server
- REST API with 24 endpoints + WebSocket for real-time updates
- Auth: login, 2FA (TOTP), session management with role-based access
- QR: activate (from camera), manual add, search, deactivate
- Batch: management, Excel ERP loader, batch history
- Devices: PLC (Omron FINS/UDP), Camera (TCP socket), Scanner (Serial port)
- Database: SQLite for QR records, users, batch history
- Config: INI file pattern (matching TApp)
- API documentation included for frontend team

## API Base URL

```
http://localhost:5000/
```
